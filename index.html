Code Fellows Goals

My primary goal in attending Code Fellows is to build my software development technical skills. I have been working with software (and hardware) developers but in order to better understand the "under the hood" technology, I wanted to get hands-on to learn the fundamentals.

My secondary goal is to better understand the world of coding bootcamps. As a transitioning veteran and someone who believes in life-long learning, I believe that careers in tech offers the best transition for veterans and I want to be able to advise and counsel those that reach out to me so I can speak more credibly; especially since federal organizations (e.g. VA) is recognizing coding bootcamps as viable transition pipeline.
